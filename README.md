## Install

1. Python dependencies:
    1. `pip install poetry`
    2. `poetry install`
2. Npm dependencies, for linter, formatter and commit linter (optional):
    1. `brew install npm`
    2. `npm install`

## Run

1. `pytest --cov=project --cov-fail-under=50 --cov-report=term-missing tests`
